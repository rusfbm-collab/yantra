### –û—Å–Ω–æ–≤–Ω–∞—è –∏–¥–µ—è: AFSM –Ω–µ–π—Ä–æ–Ω

–ù–µ–π—Ä–æ–Ω = –∞–≤—Ç–æ–º–∞—Ç (Automaton) –Ω–∞ –∫–æ–Ω–µ—á–Ω–æ–π –∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–æ–π —Å—Ç—Ä—É–∫—Ç—É—Ä–µ:

```python
class AFSMNeuron:
    def __init__(self, n: int):
        self.n = n  # –†–∞–∑–º–µ—Ä —Ü–∏–∫–ª–∏—á–µ—Å–∫–æ–π –≥—Ä—É–ø–ø—ã Z‚Çô
    
    def step(self, state):
        """–î–µ—Ç–µ—Ä–º–∏–Ω–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –ø–µ—Ä–µ—Ö–æ–¥ —Å–æ—Å—Ç–æ—è–Ω–∏—è
        
        –§–æ—Ä–º—É–ª–∞: step(o, x, i, j) = (o', x', i', j')
        –≥–¥–µ:
            o' = (o + 1) mod n      # –û—Ä–±–∏—Ç–∞
            x' = (x * i) mod n      # –í–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ
            i' = i                  # –í—Ö–æ–¥ –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è
            j' = (j * o') mod n     # –í—ã—Ö–æ–¥
        """
        o, x, i, j = state
        return (
            (o + 1) % self.n,
            (x * i) % self.n,
            i,
            (j * ((o + 1) % self.n)) % self.n
        )
```

**–ö–ª—é—á–µ–≤–æ–µ —Å–≤–æ–π—Å—Ç–≤–æ:** –î–µ—Ç–µ—Ä–º–∏–Ω–∏–∑–º - –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π –≤—Ö–æ–¥ ‚Üí –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π –≤—ã—Ö–æ–¥ (–≤—Å–µ–≥–¥–∞!).

### –û–±—É—á–µ–Ω–∏–µ: exhaustive search

–í–º–µ—Å—Ç–æ –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞ - –ø–æ–ª–Ω—ã–π –ø–µ—Ä–µ–±–æ—Ä –≤ **—Å–∂–∞—Ç–æ–º** –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ:

```python
def train(self, X, y):
    """Exhaustive search –≤ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ –∫–∞–Ω–¥–∏–¥–∞—Ç–æ–≤"""
    best_accuracy = 0.0
    best_params = None
    
    # –ì–µ–Ω–µ—Ä–∏—Ä—É–µ–º –≤—Å–µ –∫–∞–Ω–¥–∏–¥–∞—Ç—ã (–≤ —Å–∂–∞—Ç–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ)
    candidates = self.generate_candidates()  # ~64 –∫–∞–Ω–¥–∏–¥–∞—Ç–∞ –¥–ª—è XOR
    
    # –ü—Ä–æ–≤–µ—Ä—è–µ–º –∫–∞–∂–¥–æ–≥–æ
    for params in candidates:
        accuracy = self.evaluate(params, X, y)
        if accuracy > best_accuracy:
            best_accuracy = accuracy
            best_params = params
    
    self.params = best_params  # –ì–∞—Ä–∞–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ –ª—É—á—à–∏–π!
    return best_params
```

Exhaustive –ø—Ä–æ–≤–µ—Ä–∫–∞ –∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–∏—Ö —Å–≤–æ–π—Å—Ç–≤:

**–ß—Ç–æ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è:**

‚úì **–ê—Å—Å–æ—Ü–∏–∞—Ç–∏–≤–Ω–æ—Å—Ç—å:** (a ‚äó b) ‚äó c = a ‚äó (b ‚äó c) –¥–ª—è –≤—Å–µ—Ö a, b, c ‚àà Z‚Çô  
‚úì **–≠–∫–≤–∏–≤–∞—Ä–∏–∞–Ω—Ç–Ω–æ—Å—Ç—å:** step(œÉ(s)) = œÉ(step(s)) –¥–ª—è –≤—Å–µ—Ö –∞–≤—Ç–æ–º–æ—Ä—Ñ–∏–∑–º–æ–≤ œÉ  
‚úì **–ü–µ—Ä–∏–æ–¥–∏—á–Ω–æ—Å—Ç—å:** step ≥(x) = x –¥–ª—è –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–∞ —Å–∂–∞—Ç–∏—è r
